# Theme Park Management System

## Project Overview
This is a Java-based theme park management system that handles visitors, employees, rides, queues, and ride history.

## Features
- **Part 1**: Class hierarchy with Person, Employee, Visitor, and Ride classes
- **Part 2**: Abstract classes and interfaces
- **Part 3**: Queue management for ride waiting lines
- **Part 4**: LinkedList for ride history with sorting capabilities
- **Part 5**: Ride cycle simulation
- **Part 6**: File export functionality
- **Part 7**: File import functionality

## How to Run
1. Compile all Java files in the src directory
2. Run the AssignmentTwo class
3. The program will demonstrate all parts sequentially
